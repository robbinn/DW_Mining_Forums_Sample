BlackArch Linux, which is popular with many security experts, has gone a step further with the latest update. BlackArch 09/01/2021 comes with a current Linux kernel and a completely redesigned installer. What one should particularly emphasize, however, is the collection of tools that has now grown again. With no less than 2,700 tools for searching for security holes, IT forensics and much more, BlackArch will certainly continue to secure a place at the top among the most popular Linux operating systems.
